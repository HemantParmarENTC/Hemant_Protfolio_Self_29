Weight Measurement Using Chair

This project uses an Arduino Uno R3 with a Load Cell and HX711 module embedded in a chair to measure a person's weight accurately in real time. A connected display shows the reading, while a buzzer and LED provide alerts when the weight exceeds a set limit.
